# 前端开发文档

## 1. 项目概述

ClassNet 是一个专为班级内部使用设计的网络系统，提供了丰富的功能，包括视频播放、音乐播放、日历、天气、AI聊天、聊天、课程表等应用。本前端开发文档旨在详细说明前端部分的技术实现方案，确保开发人员能够按照规范进行开发。

## 2. 技术栈与限制

### 2.1 技术栈
- **基础技术**：HTML5, CSS3, JavaScript (ES5/ES6)
- **样式处理**：纯CSS + Bulma组件库
- **图标库**：FontAwesome (本地资源)
- **动画效果**：CSS动画 + JavaScript动画
- **响应式设计**：CSS Flexbox + Grid + Bulma响应式网格

### 2.2 技术限制
- 不使用任何前端框架
- 优先使用Bulma组件库，避免使用其他UI组件库
- 仅使用本地 `frontend/public/fontawesome` 目录下的FontAwesome资源
- 禁止使用iframe元素及新开标签页的操作方式
- 适配安卓8.0系统原生WebView(Chromium内核78.0版本)和安卓8.0系统原生WebView(Chromium内核89.0版本)
- 客户端为安卓横屏平板，需注意适配显示和布局
- 禁止使用浏览器原生模态框，需使用自定义模态框
- 客户端有白名单限制，只能直连服务器，无法直连外部样式或资源

## 4. UI设计规范

### 4.1 FluentUI设计语言

FluentUI设计语言强调简洁、现代、直观的用户界面。实现要点：

- **简洁的布局**：清晰的层次结构，合理的留白
- **现代的配色**：使用柔和的色调，主色调建议使用蓝色系
- **直观的交互**：明确的视觉反馈，流畅的动画效果
- **一致的组件**：统一的按钮、输入框、卡片等组件样式

### 4.2 新拟态(Neumorphism)UI风格

新拟态设计通过阴影和高光创建柔和的3D效果，实现要点：

- **柔和的阴影**：使用同色系的阴影和高光，营造凸起或凹陷的效果
- **低对比度**：背景与元素的对比度较低，整体视觉效果柔和
- **圆角设计**：使用较大的圆角，增强柔和感
- **细腻的质感**：通过微妙的阴影变化，创造细腻的质感

**CSS实现示例**：

```css
/* 新拟态按钮 */
.neumorphic-button {
  background: #e0e5ec;
  border: none;
  border-radius: 12px;
  padding: 12px 24px;
  box-shadow: 5px 5px 10px #a3b1c6, -5px -5px 10px #ffffff;
  transition: all 0.3s ease;
  cursor: pointer;
}

.neumorphic-button:hover {
  box-shadow: 3px 3px 6px #a3b1c6, -3px -3px 6px #ffffff;
}

.neumorphic-button:active {
  box-shadow: inset 2px 2px 5px #a3b1c6, inset -2px -2px 5px #ffffff;
}
```

### 4.3 玻璃拟态(Glassmorphism)设计

玻璃拟态设计通过背景模糊和透明底色，创建磨砂玻璃效果，实现要点：

- **背景模糊**：使用`backdrop-filter: blur()`实现背景模糊效果
- **透明底色**：使用半透明的背景色
- **边框效果**：添加细微的边框，增强层次感
- **光影效果**：添加微妙的阴影，增强立体感

**CSS实现示例**：

```css
/* 玻璃拟态卡片 */
.glassmorphism-card {
  background: rgba(255, 255, 255, 0.1);
  backdrop-filter: blur(10px);
  -webkit-backdrop-filter: blur(10px);
  border-radius: 16px;
  border: 1px solid rgba(255, 255, 255, 0.18);
  box-shadow: 0 8px 32px 0 rgba(31, 38, 135, 0.15);
  padding: 24px;
  transition: all 0.3s ease;
}

.glassmorphism-card:hover {
  box-shadow: 0 12px 40px 0 rgba(31, 38, 135, 0.2);
  transform: translateY(-2px);
}
```

### 4.5 Bulma组件库使用规范

Bulma是一个现代的CSS框架，提供了丰富的组件和工具类。在所有前端开发中，**必须优先使用Bulma组件库**，只有在Bulma无法满足需求时才进行自定义开发。

#### 核心使用原则

- **优先使用**：在开发任何新功能时，首先考虑使用Bulma提供的组件，如按钮、卡片、表单、导航等
- **避免重复**：不要重新发明轮子，Bulma已经提供了大量现成的组件和工具类
- **保持一致**：使用Bulma的设计系统，确保整个应用的视觉一致性
- **灵活扩展**：对于需要特殊样式的元素，可以在Bulma基础上进行自定义
- **响应式优先**：充分利用Bulma的响应式工具类，如`is-mobile`、`is-tablet`、`is-desktop`等
- **颜色系统**：使用Bulma的颜色类，如`is-primary`、`is-success`、`is-warning`、`is-danger`等
- **布局系统**：使用Bulma的网格系统进行布局，如`columns`、`column`等

#### 开发流程建议

1. **需求分析**：明确功能需求和UI设计要求
2. **组件选择**：查找Bulma文档，选择合适的组件
3. **集成实现**：直接使用Bulma组件实现功能
4. **自定义调整**：根据需要在Bulma基础上进行微调
5. **测试验证**：确保在不同设备上的兼容性

#### 重要提示

- **AI开发时**：当使用AI辅助开发时，直接要求使用Bulma组件库实现功能
- **文档参考**：Bulma官方文档提供了详细的组件使用方法和示例
- **性能优化**：合理使用Bulma组件，避免不必要的DOM元素和样式

## 6. 响应式设计

### 6.1 布局策略

- **移动端优先**：从平板（横屏）开始设计，然后扩展到移动端和桌面
- **弹性布局**：使用Flexbox和Grid实现灵活的布局
- **媒体查询**：针对不同屏幕尺寸进行优化

### 6.2 断点设置

```css
/* 媒体查询断点 */
/* 小屏幕手机 */
@media (max-width: 576px) {
  /* 样式调整 */
}

/* 平板设备 */
@media (min-width: 768px) and (max-width: 1024px) {
  /* 样式调整 */
}

/* 横屏平板（主要目标设备） */
@media (min-width: 1024px) and (orientation: landscape) {
  /* 样式调整 */
}

/* 桌面设备 */
@media (min-width: 1200px) {
  /* 样式调整 */
}
```

### 6.3 适配要点

- **横屏布局**：确保在横屏模式下布局合理，元素排列整齐
- **触摸交互**：增大点击区域，确保在触摸设备上操作流畅
- **字体大小**：使用相对单位（rem, em），确保在不同设备上字体大小合适
- **图片优化**：使用响应式图片，确保在不同设备上加载合适尺寸的图片

**JavaScript实现示例**：

```javascript
// 表单切换
const authTabs = document.querySelectorAll('.auth-tab');
const formPanels = document.querySelectorAll('.form-panel');

authTabs.forEach(tab => {
  tab.addEventListener('click', () => {
    const targetTab = tab.dataset.tab;
    
    // 更新标签状态
    authTabs.forEach(t => t.classList.remove('active'));
    tab.classList.add('active');
    
    // 更新表单状态
    formPanels.forEach(panel => {
      panel.classList.remove('active');
      panel.classList.add('hidden');
    });
    
    const targetPanel = document.getElementById(`${targetTab}-form`);
    targetPanel.classList.remove('hidden');
    setTimeout(() => {
      targetPanel.classList.add('active');
    }, 50);
  });
});

// 密码显示/隐藏
const togglePasswordButtons = document.querySelectorAll('.toggle-password');

togglePasswordButtons.forEach(button => {
  button.addEventListener('click', () => {
    const passwordInput = button.previousElementSibling;
    const icon = button.querySelector('i');
    
    if (passwordInput.type === 'password') {
      passwordInput.type = 'text';
      icon.classList.remove('fa-eye');
      icon.classList.add('fa-eye-slash');
    } else {
      passwordInput.type = 'password';
      icon.classList.remove('fa-eye-slash');
      icon.classList.add('fa-eye');
    }
  });
});

// 表单验证
const forms = document.querySelectorAll('form');

forms.forEach(form => {
  form.addEventListener('submit', (e) => {
    e.preventDefault();
    
    if (validateForm(form)) {
      // 显示加载状态
      document.getElementById('loading-overlay').classList.add('active');
      
      // 模拟API调用
      setTimeout(() => {
        document.getElementById('loading-overlay').classList.remove('active');
        // 处理登录/注册逻辑
      }, 1500);
    }
  });
  
  // 实时验证
  const inputs = form.querySelectorAll('input');
  inputs.forEach(input => {
    input.addEventListener('input', () => {
      validateField(input);
    });
  });
});

function validateForm(form) {
  const inputs = form.querySelectorAll('input[required]');
  let isValid = true;
  
  inputs.forEach(input => {
    if (!validateField(input)) {
      isValid = false;
    }
  });
  
  return isValid;
}

function validateField(input) {
  const errorMessage = input.nextElementSibling;
  let isValid = true;
  let message = '';
  
  // 验证逻辑
  if (input.id === 'register-password') {
    if (input.value.length < 8) {
      message = '密码长度至少8位';
      isValid = false;
    }
  }
  
  if (input.id === 'register-confirm-password') {
    const passwordInput = document.getElementById('register-password');
    if (input.value !== passwordInput.value) {
      message = '两次输入的密码不一致';
      isValid = false;
    }
  }
  
  // 显示错误信息
  if (errorMessage) {
    errorMessage.textContent = message;
    errorMessage.style.display = isValid ? 'none' : 'block';
  }
  
  return isValid;
}
```

// API基础配置
const API_BASE_URL = 'http://localhost:8000/api';

// 通用请求方法
async function request(endpoint, options = {}) {
  const url = `${API_BASE_URL}${endpoint}`;
  
  const defaultOptions = {
    headers: {
      'Content-Type': 'application/json',
    },
  };
  
  // 添加认证令牌
  const token = localStorage.getItem('token');
  if (token) {
    defaultOptions.headers['Authorization'] = `Bearer ${token}`;
  }
  
  const mergedOptions = {
    ...defaultOptions,
    ...options,
    headers: {
      ...defaultOptions.headers,
      ...options.headers,
    },
  };
  
  try {
    const response = await fetch(url, mergedOptions);
    
    if (!response.ok) {
      const errorData = await response.json().catch(() => ({}));
      throw new Error(errorData.message || `请求失败: ${response.status}`);
    }
    
    return await response.json();
  } catch (error) {
    console.error('API请求错误:', error);
    throw error;
  }
}

export function fadeIn(element, duration = 300) {
  element.style.opacity = '0';
  element.style.display = 'block';
  
  let start = null;
  function animate(timestamp) {
    if (!start) start = timestamp;
    const progress = timestamp - start;
    const opacity = progress / duration;
    
    element.style.opacity = opacity;
    
    if (progress < duration) {
      requestAnimationFrame(animate);
    }
  }
  
  requestAnimationFrame(animate);
}

// 动画效果：淡出
export function fadeOut(element, duration = 300) {
  let start = null;
  function animate(timestamp) {
    if (!start) start = timestamp;
    const progress = timestamp - start;
    const opacity = 1 - (progress / duration);
    
    element.style.opacity = opacity;
    
    if (progress < duration) {
      requestAnimationFrame(animate);
    } else {
      element.style.display = 'none';
    }
  }
  
  requestAnimationFrame(animate);
}

export function slideIn(element, direction = 'left', duration = 300) {
  const startPositions = {
    left: { transform: 'translateX(-100%)' },
    right: { transform: 'translateX(100%)' },
    top: { transform: 'translateY(-100%)' },
    bottom: { transform: 'translateY(100%)' }
  };
  
  element.style.opacity = '0';
  element.style.transform = startPositions[direction].transform;
  element.style.display = 'block';
  
  setTimeout(() => {
    element.style.transition = `all ${duration}ms ease`;
    element.style.opacity = '1';
    element.style.transform = 'translate(0)';
  }, 10);
}

// 基于姓名生成颜色
export function getColorFromName(name) {
  const colors = ['#3B82F6', '#10B981', '#F59E0B', '#EF4444', '#8B5CF6', '#EC4899'];
  let hash = 0;
  
  for (let i = 0; i < name.length; i++) {
    hash = name.charCodeAt(i) + ((hash << 5) - hash);
  }
  
  const index = Math.abs(hash) % colors.length;
  return colors[index];
}

// 防抖函数
export function debounce(func, wait) {
  let timeout;
  return function executedFunction(...args) {
    const later = () => {
      clearTimeout(timeout);
      func(...args);
    };
    clearTimeout(timeout);
    timeout = setTimeout(later, wait);
  };
}

// 节流函数
export function throttle(func, limit) {
  let inThrottle;
  return function executedFunction(...args) {
    if (!inThrottle) {
      func.apply(this, args);
      inThrottle = true;
      setTimeout(() => inThrottle = false, limit);
    }
  };
}
